### 对象继承的实现

* 我们发现一个非常有意思的地方

当我们用一个对象和一个数字或字符串类型的数字做比较式，JS会自动调用对象内置的toString方法对对象进行转化，这个是非常有意思的地方，从中也能看出JS在做一般比较时的运行机制，通常会将对象转化为一个最基本类型的值来做比较。

下面的代码虽然在实际运用中没有什么意义，同时也不鼓励大家这样随意改造对象继承来的初始方法。

```javascript

// 对象内置方法
let test = {a: 1, toString () {console.log(this.a++)}}

if (test == 1 || test == 2) {
  console.log(3)
}

console.log(test.a);

```
